La prueba tecnica de back se realizo con: Xampp, MySQL, GraphQL (Manejada con la extension de google) 

1: Crear la base de datos en MySQL con las querys dadas (archivo sql_createDB)
2: Extraer la carpeta en la carpeta httdocs del servidor local Xampp
3: Iniciar servidor Xampp
4: Direccionar a la carpeta extraida -> http://localhost:YOUR_PORT/pruebaBack/
5: Copiar url del servidor con la direccion de la carpeta extraida e ir a graphQL (extension de google) y en el endpoint copear esta url
6: Listo ya se puede probar y realizar peticiones
